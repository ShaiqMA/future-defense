# future-defense


## Over de Game

**Future Defense: Final Launch** is een top-down wave-based shooter, ontwikkeld voor **Best Education B.V.**.
Het doel: verdedigen van je lanceerplatform tegen waves van vijanden.

**Branding aanwezig:**

* Naam: Best Education
* Logo: weergegeven in start- en gameplay
* Slogan: “Wij lanceren je de toekomst in!”

**Functionaliteiten:**

* Start, Win en Lose states
* Player besturing (WASD)
* Schieten met Space
* Waves met toenemende moeilijkheid
* Score bijhouden
* Restart knop

 Installatie en Starten

### 1 Project openen

1. Open **Unity Hub**
2. Klik op **Add**, selecteer de map `FutureDefense_FinalLaunch`
3. Open het project

### 2️ Scene openen

1. In **Assets → Scenes**, dubbelklik op:

```
MainScene
```

2. Zorg dat deze scène open is

### 3️ Spel starten

1. Druk **Play** in Unity
2. Klik **Start** op het StartPanel

### 4️ Speler besturen

* **Beweging:** W, A, S, D of pijltjestoetsen
* **Schieten:** Spacebar

### 5️ Gameplay

* Defend het lanceerplatform tegen waves
* Elke wave wordt moeilijker (meer en snellere vijanden)
* Score wordt automatisch bijgehouden

### 6️ Win / Lose

* Win: alle waves voltooid
* Lose: health 0
* Klik **Restart** om opnieuw te spelen
